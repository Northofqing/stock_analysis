## 5. 验证 / Completion Rule

CLAUDE.md Completion Rule 三层 — 每 commit 必须:

```bash
# 1. 模块层 (模块代码 + cargo test --lib + cargo build --lib exit 0)
cargo test --lib <module>::
cargo build --lib

# 2. 集成 grep (主 binary 实际引用)
grep -RInE 'use stock_analysis::<module>|<module>::' src/bin/monitor/ --include='*.rs' | wc -l
# 期望: ≥ 1 hit per new module

# 3. Live binary (cargo build --release + --test)
cargo build --release --bin monitor
./target/release/monitor --test 2>&1 | grep -E '<new_module_keywords>' | head -5
# 期望: ≥ 1 hit (模块实际跑通)
```

零 hits = 集成 0% = NOT complete。
