# v15.3 Phase D 扩展计划 — 全量新闻 + 实盘/政策/财报/机构观点

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans. Steps use checkbox (`- [ ]`) syntax.

**Goal:** 把 Phase D 从"全量新闻搜索 + 股票分析"扩展为 4 路数据源整合：
1. **新闻** (Phase D 原计划, 19 个 flash provider)
2. **实盘情况** (新增, 今日持仓/成交/盈亏/账户模式)
3. **政策** (新增, 十五五规划/发改委/工信部/国务院)
4. **财报** (新增, EM F10 + 公告流 + 业绩预告)
5. **机构观点** (新增, 卖方共识 + 行业策略报告)

**Architecture:**
- 4 路数据源走统一 `NewsFeed` trait → `NewsAggregator` → `EntityExtractor` (复用 `event_extractor`) → `StockMapper` (复用 `chain_registry` + `bom_kb`) → `ImpactAnalyzer` → `NewsDispatcher` → v14 push 栈
- 复用现有: `src/opportunity/event_extractor/` (promote), `data_provider::financials`, `data_provider::consensus`, `data_provider::industry`, `data_provider::announcement`

let score = (base + dir_bonus + imp_bonus + source_weight) * decay * confidence;
