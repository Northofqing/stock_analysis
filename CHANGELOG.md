## [Unreleased] - 2026-02-04

### 🎉 新增功能: 多因子量化选股与回测系统

#### 核心功能
- ✅ **多因子选股策略**: 基于市值/ROE/PE/PB/换手率的量化选股
- ✅ **回测引擎**: 完整的交易模拟 + 性能指标计算
- ✅ **自动化集成**: 股票分析完成后自动执行回测
- ✅ **结果推送**: 回测结果自动包含在邮件通知中

#### 多因子策略
**支持的因子:**
- 市值因子 (MarketCap) - 小市值优先 (权重 20%)
- ROE 因子 - 高 ROE 优先 (权重 30%)
- PE 因子 - 低 PE 优先 (权重 20%)
- PB 因子 - 低 PB 优先 (权重 15%)
- 换手率因子 - 高换手率优先 (权重 15%)

**选股流程:**
1. 收集所有已分析股票的因子数据
2. 对每个因子进行排名（0-100分）
3. 根据权重加权计算总分
4. 选择得分最高的 Top N 股票

#### 回测引擎
**支持功能:**
- 交易模拟（买入/卖出/持仓管理）
- 成本计算（滑点 0.1% + 手续费 0.03%）
- 定期调仓（默认 30 天）
- 每日市值记录
- 性能指标计算

**性能指标:**
- 总收益率 (Total Return)
- 年化收益率 (Annual Return)  
- 最大回撤 (Max Drawdown)
- 夏普比率 (Sharpe Ratio)
- 总交易次数
- 胜率 (Win Rate)

## [Unreleased] - 2025-01-27

### ⭐ 新增功能: 定时任务

#### 核心功能
- ✅ **间隔执行模式**: 按固定时间间隔循环执行（如每30分钟、每2小时）
- ✅ **指定时间点模式**: 每天在固定时间执行（支持多个时间点，如09:15,15:30）
- ✅ **工作日限制**: 只在指定星期执行（如1-5仅工作日，跳过周末）
- ✅ **立即执行选项**: `--run-now` 启动时立即执行一次，无需等待首个定时点
- ✅ **智能时间计算**: 自动跳过已过时间，计算下一个最近执行时间
- ✅ **多时间点支持**: 一天内可设置多个执行时间（如开盘前+收盘后）

#### 命令行参数
```bash
--schedule              # 启用定时任务模式
--schedule-time <TIME>  # 指定时间点 (格式: HH:MM 或 "HH:MM,HH:MM")
--interval <MINUTES>    # 执行间隔 (分钟)
--weekdays <DAYS>       # 指定星期 (逗号分隔, 1=周一, 7=周日)
--run-now               # 立即执行一次，不等待定时
```

#### 使用示例
```bash
# 每30分钟执行一次
cargo run --release -- --schedule --interval 30

# 每天9:30执行
cargo run --release -- --schedule --schedule-time "09:30"

# 工作日两次执行（开盘前+收盘后）
cargo run --release -- --schedule \
  --schedule-time "09:15,15:30" \
  --weekdays 1,2,3,4,5 \
  --run-now
```

#### 文档
- [定时任务完整指南](SCHEDULE_GUIDE.md)
- [README 新增章节](README.md#⏰-定时任务模式)

#### 技术实现
- 使用 `chrono` 库进行时间计算和日期处理
- 使用 `tokio::time::sleep` 实现异步等待
- 三种定时模式共存，灵活切换
- 完善的错误处理和日志输出
- 支持 Ctrl+C 优雅退出

#### 盈利指标集成
- ✅ **市盈率 (PE)**: 估值水平判断（高估/合理/低估）
- ✅ **市净率 (PB)**: 资产估值分析

- AI分析新增5个维度：
  1. **估值分析**: PE/PB综合评估
  2. **市值分析**: 市场地位和流动性
  3. **流动性分析**: 换手率和流通比例
  4. **活跃度评估**: 交易热度
  5. **综合研判**: 多维度总结
